LJCCityList ReadMe

File Name Description
LJCCityList.html The web app entry page.
Displays a paged list of cities and provides New, Edit and Delete capabilities.
LJCCityListEvents.js Creates the main LJCCityList.html event listeners and contains the event handler methods.
CityList/LJCCityTableEvents.js Creates the city HTML table event listeners and contains the event handler methods.
CityList/LJCCityPageData.js The data for LJCCityTable.php.
City/LJCCityTable.php Retrieves the City data and builds the City HTML table text. The table text is returned to the caller to be added to the HTML Document.
LJCTableData.js Contains a helper class designed to be associated with an HTML table and menu.
It contains common HTML table helper methods and properties
LJCDataConfigs.php Retrieves the data configuration from the DataConfigs.xml file.
LJCRoot.php Contains the method RelativePrefix() which establishes the include file relative path from the current code line.
CSS/Table.css The LJCCityList.html CSS file.
