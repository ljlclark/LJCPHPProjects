Initial "List" Page

An LJC Maintenance web app contains a dynamically created HTML Table "List" for displaying and paging through the main source or table items. In this document it is often referred to as a "Maint" app.

The page contains controls for selecting the parent record if the table has a parent table or tables. The list displays the items that belong to the selected parent record.

The maint app name uses the name of the main item source or table as in: "LJCCity".

The initial list page of a maint app is named for the app as in: "LJCCityList.html"

The parent tables usually have separate maint apps to reduce complexity.

List Events Script

The list page event initialization and handlers are defined in a separate JavaScript file that is named for the list page as in: "LJCCityListEvents.js".

This file handles common events for the list page. However, HTML Tables and other significant items that have many events or large event handlers are defined in separate files.

The list events class is named "LJCCityListEvents".

The list events class contains a reference to an LJCTable object named CityTable.

The list events class creates the table events script and contains the reference object named CityTableEvents.

The LJCTable object

The LJCTable object contains the state of the HTML table such as the current row index and paging state such as the beginning and end of data flags and the current row keys.

The LJCTable object also contains functions for accessing and manipulating HTML table elements.

Table Events Script

The event initialization and handlers for the list HTML table are defined in a separate JavaScript file that is named for the list HTML table as in: "LJCCityTableEvents.js".

The table events class constructor requires several parameters. It takes an LJCCityTableEventsArgs object with the main parameter values, instead of receiving each parameter separately

The table events file handles table paging, context menu display, context menu item selection and other functions directly related to the list HTML table.

The table events class is named "LJCCityTableEvents".

The private method "#AddEvents()" adds the events for displaying the context menu and handling the menu items.

The list HTML table is created dynamically. Therefore, the HTML table specific events must be recreated each time the HTML table is created.
A separate private method "#AddTableEvents()" exists for this purpose.

Table Web Service

The table web service is a PHP web service that creates HTML table text from a database table.

For a maint app, it is called from the table events object.

Detail Events Script

The event initialization and handlers for the detail dialog are defined in a separate JavaScript file named LJCCityDetailEvents.js.

The detail events class is named "LJCCityDetailEvents".

Data Web Service

The data web service is a PHP web service for accessing a database table. It provides Add, Delete, Retrieve and Update functionality.

For a maint app, it is generally called from the detail events object.
