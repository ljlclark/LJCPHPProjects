Class LJCCityListEvents

Creates the LJCCityList.html event listeners and contains the event handlers.

Constructor Methods
constructor Initializes the object instance.
AddEvents Creates the LJCCityList.html event listeners.

Document Event Handlers
DocumentContextMenu Displays the City context menu.
DocumentDoubleClick Performs the "Edit" menu selection.
DocumentKeyDown The Document "keydown" event handler.

Menu Event Handlers
Delete Deletes the selected item.
Edit Displays a City Detail dialog for editing the selected item.
New Displays a City Detail dialog for creating a new City item.
Next Displays the next list page items if data is available.
Previous Displays the previous list page items if data is available.
Refresh Reloads the current list page.

Other Methods

SubmitDetail Submit the hidden form data to CityDetail.php.
