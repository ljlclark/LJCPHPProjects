Class Cities

Represents a collection of city data objects.

Collection Methods
Add Creates and adds the item to the list.
AddObject Adds the supplied city to the list.
Clear Clears the collection list.
GetIndex Gets the index of the item with the supplied name.
Items Gets the items that match the supplied names.
Names Gets an array of names.
Remove Removes the item with the supplied name.
Retrieve Retrieves the item with the supplied name.
RetrieveWithIndex Retrieves the item at the supplied index.
