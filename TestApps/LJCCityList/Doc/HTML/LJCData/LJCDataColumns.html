Class LJCDataColumns

Represents a collection of data column definitions.

Static Methods
AddValue Adds the provided value to the item with the provided propertyName.

Methods
Add Creates and adds the column object to the list.
AddObject Adds the supplied column to the list.
Clear Clears the collection list.
Columns Gets the column objects that match the property names.
GetIndex Gets the index for the item with the supplied property name.
PropertyNames Gets an array of property names.
Remove Removes the column object with the supplied property name.
Retrieve Retrieves the column object with the supplied property name.
RetrieveWithIndex Retrieves the column object with the supplied index.
